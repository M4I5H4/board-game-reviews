# 📊 Rating Criteria Explained

- **Fun factor** → Pure enjoyment & engagement during play.  
- **Visual design** → Artwork, board layout, cards, graphic design appeal.  
- **Component quality** → Durability and tactile feel of pieces/cards/board.  
- **Rulebook** → Clarity, examples, accessibility of rules.  
- **Theme** → How well the theme is woven into gameplay.  
- **Replay value** → Variety and longevity over multiple plays.  
- **Length** → Perceived appropriateness of setup & playtime vs payoff.  
- **Player interaction** → Negotiation, trading, take-that, cooperation, etc.  
- **Strategic value** → Depth of decisions, meaningful choices.  
- **Luck factor** → Balance between skill and randomness (dice/cards/etc.).
